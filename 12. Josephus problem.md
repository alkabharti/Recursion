### Solution :

Time Complexity :


```java
public int josephus(int n, int k)
{
    int res=0;
    int i;
    for(i=1;i<=n;i++)
    {
        res=(res+k)%i;
    }    
    return res+1;    
}
```



```java
public int josephus(int n, int k)
{
    if (n == 1)
        return n;
    return (josephus(n - 1, k) + (k - 1)) % n + 1;
}
```
